# Швидка довідка по командам адміністратора

## 📋 Список команд

### 1. `/users` - Список користувачів
Показує всіх зареєстрованих користувачів з їх статусом.

**Використання:**
```
/users
```

**Приклад виводу:**
```
Список зареєстрованих користувачів:
• `123456789` (@john_doe) — вільний
• `987654321` (@jane_smith) — в чаті з 111222333
• `111222333` (@bob_jones) — в чаті з 987654321
```

---

### 2. `/connect ID1 ID2` - З'єднати користувачів
З'єднує двох користувачів для приватного спілкування.

**Використання:**
```
/connect 123456789 987654321
```

**Що відбувається:**
1. Користувачі з'єднуються
2. Адміністратор отримує підтвердження
3. Обидва користувачі отримують сповіщення про підключення
4. Тепер вони можуть обмінюватися повідомленнями через бота

**Повідомлення користувачам:**
```
Адміністратор підключив вас до розмови. Можете спілкуватися! 😉
```

---

### 3. `/disconnect ID` - Роз'єднати користувача
Роз'єднує користувача та його партнера.

**Використання:**
```
/disconnect 123456789
```

**Що відбувається:**
1. Користувач та його партнер роз'єднуються
2. Обидва отримують сповіщення про завершення діалогу
3. Вони не можуть більше обмінюватися повідомленнями

**Повідомлення користувачам:**
```
Ваш діалог було завершено адміністратором. ❌
```

---

### 4. `/history ID [кількість]` - Історія користувача
Показує всі повідомлення (відправлені + отримані) конкретного користувача.

**Використання:**
```
/history 123456789           # Останні 20 повідомлень
/history 123456789 50        # Останні 50 повідомлень
/history 123456789 100       # Останні 100 повідомлень (максимум)
```

**Приклад виводу:**
```
📜 Історія повідомлень користувача `123456789` (останні 20):

➡️ 17.07.2026 14:30 | Text
   Привіт! Як справи?

⬅️ 17.07.2026 14:32 | Photo
   Подивись на це фото!

➡️ 17.07.2026 14:35 | Voice

⬅️ 17.07.2026 14:40 | Sticker
```

**Позначення:**
- ➡️ = відправлено користувачем
- ⬅️ = отримано користувачем

---

### 5. `/chat ID1 ID2 [кількість]` - Переписка двох користувачів
Показує переписку між двома конкретними користувачами.

**Використання:**
```
/chat 123456789 987654321           # Останні 50 повідомлень
/chat 123456789 987654321 100       # Останні 100 повідомлень
/chat 123456789 987654321 200       # Останні 200 повідомлень (максимум)
```

**Приклад виводу:**
```
💬 Переписка між `john_doe` та `jane_smith` (останні 50):

👤 **john_doe** (17.07 14:30)
   📋 Text
   💬 _Привіт! Як справи?_

👥 **jane_smith** (17.07 14:32)
   📋 Photo
   💬 _Подивись на це фото!_

👤 **john_doe** (17.07 14:35)
   📋 Voice

👥 **jane_smith** (17.07 14:40)
   📋 Sticker
```

**Позначення:**
- 👤 = перший користувач (ID1)
- 👥 = другий користувач (ID2)

---

## 🔍 Різниця між /history та /chat

### `/history ID`
- Показує **ВСІ** повідомлення користувача
- Включає спілкування з **різними** партнерами
- Корисно для аналізу активності користувача

**Приклад:**
Користувач 123 спілкувався з користувачем 456 та 789.
`/history 123` покаже повідомлення і з 456, і з 789.

### `/chat ID1 ID2`
- Показує **ТІЛЬКИ** переписку між двома конкретними користувачами
- Фокус на діалозі між парою
- Корисно для перегляду конкретної розмови

**Приклад:**
`/chat 123 456` покаже тільки діалог між 123 та 456.
Повідомлення користувача 123 до 789 **не** будуть показані.

## 📊 Типи повідомлень

При перегляді історії ви побачите наступні типи:

| Тип | Опис |
|-----|------|
| `Text` | Текстове повідомлення |
| `Photo` | Фотографія |
| `Voice` | Голосове повідомлення |
| `Audio` | Аудіофайл |
| `Video` | Відео |
| `Document` | Документ |
| `Sticker` | Стікер |
| `VideoNote` | Кружечок (відеоповідомлення) |

## 💡 Поради

### 1. Швидкий аналіз активності
```
/users                    # Подивіться, хто онлайн
/history 123456789 10     # Швидко перевірте останні 10 повідомлень
```

### 2. Глибокий аналіз переписки
```
/chat 123456789 987654321 200   # Отримайте повну картину діалогу
```

### 3. Знайти партнера користувача
```
/users                    # Знайдіть ID партнера в списку
/chat USER_ID PARTNER_ID  # Перегляньте їх переписку
```

### 4. Моніторинг підозрілої активності
```
/history SUSPICIOUS_USER_ID 100   # Перевірте всі повідомлення
```

---

## ⚠️ Обмеження

- `/history` - максимум **100** повідомлень за раз
- `/chat` - максимум **200** повідомлень за раз
- Повідомлення відображаються у **хронологічному порядку** (від старих до нових)
- Текст обрізається до **50 символів** в `/history` та **100 символів** в `/chat`

---

## 🚀 Типові сценарії використання

### Сценарій 1: Нові користувачі
```
1. Користувач пише боту /start
2. Адміністратор отримує сповіщення в консолі
3. /users                           # Дивимося список
4. /connect NEW_USER_ID PARTNER_ID  # З'єднуємо
```

### Сценарій 2: Скарга на користувача
```
1. Отримали скаргу на користувача 123
2. /history 123 50                  # Перевіряємо його історію
3. /users                           # Дивимося з ким він спілкується
4. /chat 123 PARTNER_ID 100         # Перевіряємо конкретний діалог
5. /disconnect 123                  # При потребі роз'єднуємо
```

### Сценарій 3: Завершення розмови
```
1. Користувачі закінчили спілкування
2. /disconnect USER_ID              # Роз'єднуємо пару
3. /users                           # Підтверджуємо що обидва "вільні"
```

---

## 🔐 Безпека

- Тільки адміністратори (з `AdminChatIds`) можуть використовувати ці команди
- Звичайні користувачі НЕ бачать чужих повідомлень
- Всі дії логуються в консолі сервера
- Історія зберігається в захищеній базі даних
